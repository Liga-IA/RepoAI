# Source Code: The Categorization Engine

This directory contains the Python implementation of the **SCA-Lex Hybrid Fusion** core pipeline.

### Overview
The `visualize_scientific_split.py` script handles the mathematical projection of image embeddings onto the unit hypersphere. By computing the Cosine Similarity between the visual evidence and our `Tuned Forensic Centroids`, it successfully routes evidence to the correct priority folders.

### Execution
Run this script from the root directory:
```bash
python src/visualize_scientific_split.py
```
