# Welcome to Repo AI 🚀

This repository is a centralized hub for Artificial Intelligence knowledge, created to share tutorials, code, projects, and study materials within the UFSC community.

Our goal is to organize AI content in a clear, scalable, and accessible way for students, researchers, and developers.

## 🗂️ What You'll Find Here

- 📊 Classical Machine Learning algorithms
- 🤖 Reinforcement Learning experiments
- 👁️ Computer Vision projects
- 🧠 LLM and NLP studies
- ⚙️ Optimization and engineering problems
- 🛠️ Templates and reusable code

## 📚 Repository Organization

Repositories follow the pattern:
category_code + _ + topic

| Category code | Area |
|------|------|
| `ml` | Machine Learning |
| `dl` | Deep Learning |
| `cv` | Computer Vision |
| `nlp` | Natural Language Processing |
| `rl` | Reinforcement Learning |
| `opt` | Optimization |
| `sys` | Systems / Embedded |
| `res` | Research / Experiments |
| `util` | Utilities / Templates |
