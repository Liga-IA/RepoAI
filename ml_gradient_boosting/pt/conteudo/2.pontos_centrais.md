# Pontos centrais de Gradient Boosting

## Introdução

Gradient boosting é um modelo de ensemble construído através de vários aprendizes fracos, modelos preditivos simples ou básicos, que tentam corrigir os erros e resíduos dos anteriores de modo que, ao fim do treinamento, é formado um modelo forte, robusto e confiável para a tarefa em questão.
Os aprendizes podem variar desde redes neurais simples, como MLPs rasas, até árvores de decisão rasas e simples, de baixa profundidade, sendo elas as mais comuns para modelos de gradient boosting, pois são simples e são aplicáveis em quase todas as situações.

## Funcionamento

A cada iteração dos aprendizes, é calculado os pseudo-resíduos (gradiente negativo da perda) e então um novo aprendiz é treinado a fim de aproximá-los, ao fim, todos são combinados de modo a formar o modelo final, seja baseado num número estabelecido e iterações ou numa métrica alvo a ser atingida (early stopping). Além disso, cada aprendiz é somado ao modelo com um coeficiente; o parâmetro learning_rate (shrinkage) controla essa contribuição, ajudando também a reduzir overfitting quando bem ajustado.

## Vantagens

Dentre as principais vantagens do gradient boosting, pode-se citar seu bom desempenho em dados tabulares e à interpretabilidade relativa, é uma ótima escolha quando os dados são tabulares e/ou o volume de dados não justifica uma grande rede neural. Ou até em casos com poucos dados que são bem rotulados e definidos.

## Desvantagens

Entretanto, quando se trata de dados mais complexos e menos tabelados, o gradient boosting peca em performance e resultados, além de requisitar tuning para apresentar os melhores resultados e evitar overfitting, embora variantes modernas possuam otimizações para estes problemas.

## Componentes

Dentre os principais componentes e parâmetros utilizados para a construção de um modelo de gradient boosting, pode-se citar: learning rate (quanto cada aprendiz afeta o modelo), n_estimators (número de iterações), max_depth (profundidade dos aprendizes), subsample (fração de amostras usadas no treino), etc.
